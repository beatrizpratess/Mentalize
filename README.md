# Mentalize — Plataforma de Cursos Online

Mentalize é uma plataforma de ensino digital desenvolvida com o objetivo de oferecer cursos acessíveis e de qualidade para impulsionar o crescimento profissional e acadêmico dos usuários.

A Mentalize busca democratizar o conhecimento por meio de uma interface simples, moderna e intuitiva.

---

## Funcionalidades

* Catálogo de cursos (Publicidade, Design, Programação e Pacote Office)
* Sistema de carrinho para matrícula
* Área do aluno (perfil e cursos em andamento)
* Sistema de login e cadastro
* Formulário de inscrição nos cursos
* Interface moderna e responsiva

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
